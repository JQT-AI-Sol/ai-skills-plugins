# 見積書フォーマット仕様

## ヘッダー情報

| フィールド | 内容 | 例 |
|-----------|------|-----|
| タイトル | 「御見積書」 | 固定 |
| 見積番号 | EST-YYYYMMDD-NNN | EST-20260311-001 |
| 見積日 | 発行日 | 2026年3月11日 |
| 有効期限 | 見積日から30日（デフォルト） | 2026年4月10日 |

## 宛先

| フィールド | 内容 |
|-----------|------|
| 顧客名 | 「○○株式会社 御中」 |
| 部署名 | 任意 |
| 担当者名 | 「○○ 様」 |

## 件名

| フィールド | 内容 | 例 |
|-----------|------|-----|
| 件名 | 案件の正式名称 | 「建設書類自動化システム開発」 |

## 合計金額（ヘッダー表示）

| フィールド | 内容 |
|-----------|------|
| 合計金額 | 税込合計を大きく表示「¥X,XXX,XXX-（税込）」 |

## 明細テーブル

| 列 | 内容 | 備考 |
|----|------|------|
| No. | 連番 | 1, 2, 3... |
| 項目名 | 作業項目 | 例: 「要件定義」 |
| 数量 | 数量 | 例: 1 |
| 単位 | 「式」「人月」「時間」等 | |
| 単価 | 円 | ¥XXX,XXX |
| 金額 | 数量×単価 | ¥XXX,XXX |

### 標準項目テンプレート（請負型・システム開発）

小規模案件（100万〜150万円）:
1. 要件定義・設計
2. フロントエンド開発
3. バックエンド開発
4. テスト・品質管理
5. 納品・導入支援

中規模案件（150万〜300万円）:
1. 要件定義
2. 基本設計
3. 詳細設計
4. フロントエンド開発
5. バックエンド開発
6. データベース構築
7. API連携開発
8. テスト・品質管理
9. 納品・導入支援
10. ドキュメント作成

### AI関連オプション項目
- AI/LLM API連携開発
- プロンプトエンジニアリング
- AIモデルチューニング
- AI機能PoC検証

## 小計・税・合計

| フィールド | 計算 |
|-----------|------|
| 小計 | 明細合計 |
| 消費税（10%） | 小計 × 0.10 |
| 合計 | 小計 + 消費税 |

## 備考欄

デフォルト記載事項:
```
【お支払条件】
着手金: 契約締結後、着手金として見積金額の50%を請求
残金: 納品・検収完了後、残額を請求
お支払期限: 請求書発行月の翌月末日

【納期】
契約締結後 約○週間（個別契約にて確定）

【前提条件】
・本見積は上記スコープに基づくものです
・スコープ変更が生じた場合は別途協議のうえ再見積とします
・インフラ費用（Vercel, Supabase等）は別途月額費用が発生します

【有効期限】
本見積書の有効期限は発行日より30日間とします
```

## 別紙：スコープ定義書（シート2・必須）

見積書には必ず「別紙_スコープ定義」シートを付ける。

### レイアウト

```
A1: 別紙：スコープ定義書
A2: {案件名} — 御見積書 {見積番号} 対応

A4: ■ 本見積に含まれる機能（スコープ内）
A5ヘッダー: No. / 機能カテゴリ / 機能名 / 概要 / 対応フェーズ
A6〜: 機能一覧（requirements.md の F-xxx から抽出）

（空行）

■ 本見積に含まれない機能（スコープ外）
ヘッダー: No. / 機能カテゴリ / 機能名 / 備考（除外理由） / 対応フェーズ
機能一覧（requirements.md の Out of Scope から抽出）

（空行）

■ 判明している要件・前提条件
ヘッダー: No. / カテゴリ / 要件・前提 / 備考 / ステータス
要件一覧（技術スタック、業務前提、運用条件など）
ステータスは「確定」or「仮説（要検証）」
```

### スタイル
- セクションタイトル（■行）: 太字、背景なし
- テーブルヘッダー: #1B3A5C背景、白文字（御見積書シートと統一）
- テーブル本文: メイリオ 10pt、罫線あり
- 「対応フェーズ」列: PoC / 本開発 / 将来拡張 など案件に応じて設定

## スタイル仕様（xlsx）

### カラー
- ヘッダー背景: #1B3A5C（濃紺）
- ヘッダー文字: #FFFFFF（白）
- 合計行背景: #F0F4F8（薄いグレー）
- 罫線: #333333
- 金額強調: #1B3A5C（濃紺）太字

### フォント
- 見出し: メイリオ or Yu Gothic, 14pt 太字
- 本文: メイリオ or Yu Gothic, 10pt
- 金額: メイリオ or Yu Gothic, 12pt 太字

### レイアウト
- A4縦（ランドスケープではない）
- 印刷範囲設定済み
- ページ余白: 上下左右 2cm
