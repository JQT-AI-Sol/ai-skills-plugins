# リーガルチェック チェックリスト

## 使い方
対象契約書を全文読み込んだ後、以下のカテゴリを順にチェックする。
各項目を GREEN / YELLOW / RED で判定し、RED は必ず修正案を提示する。

---

## A. 契約構造・基本事項

| ID | チェック項目 | RED基準 |
|----|-------------|---------|
| A1 | 当事者の特定（甲乙の定義が明確か） | 当事者が不明確、または一方的な定義 |
| A2 | 契約類型の明示（請負/準委任/混合） | 類型の記載なし、または曖昧 |
| A3 | 個別契約との関係（優先順位） | 基本契約と個別契約の優先関係が不明確 |
| A4 | 有効期間・自動更新条件 | 更新拒絶の通知期間が不合理（例: 3日前） |
| A5 | 準拠法・管轄裁判所 | 管轄条項なし |

## B. 業務遂行・納品

| ID | チェック項目 | RED基準 |
|----|-------------|---------|
| B1 | 業務範囲の定義 | 範囲が無限定、またはスコープクリープのリスク |
| B2 | 納入物・検収条件（請負の場合） | 検収基準が不明確、検収期間が短すぎる（5日未満） |
| B3 | 検収みなし条件 | みなし規定なし（甲が検収しないリスク） |
| B4 | 報告義務 | 過度な報告義務（偽装請負リスク） |
| B5 | 再委託 | 全面禁止で例外なし |

## C. 対価・支払

| ID | チェック項目 | RED基準 |
|----|-------------|---------|
| C1 | 支払条件・時期 | 支払時期が不明確、または著しく長い（90日超） |
| C2 | 請負/準委任それぞれの精算方法 | 精算方法の記載なし |
| C3 | 遅延損害金 | 遅延損害金の規定なし |
| C4 | 振込手数料の負担 | 負担者の明記なし |
| C5 | 着手金・中間金 | 大型案件で着手金なし |

## D. 知的財産権（IP）

| ID | チェック項目 | RED基準 |
|----|-------------|---------|
| D1 | 著作権の帰属（著作権法27条・28条含む） | 著作権が一方的に全帰属（対価の考慮なし） |
| D2 | 既存著作物の留保 | 既存著作物の扱いが未定義 |
| D3 | ライセンス条件（非独占/独占、範囲） | ライセンス範囲が無制限 |
| D4 | 産業財産権の帰属 | 帰属が一方的で対価なし |
| D5 | AI生成物の扱い | AI利用に関する規定なし |
| D6 | 著作者人格権の不行使 | 不行使条項なし |
| D7 | 残留知識（Residuals）条項 | 一般的知識・経験の利用制限が過度 |
| D8 | 第三者知的財産権の保証 | 保証範囲が無限定 |

## E. 契約不適合・責任

| ID | チェック項目 | RED基準 |
|----|-------------|---------|
| E1 | 契約不適合責任の期間 | 期間が1年超、または無期限 |
| E2 | 契約不適合の定義 | 定義が不明確 |
| E3 | 修補の範囲・方法 | 修補義務が無限定 |
| E4 | 甲の修正後の免責 | 甲修正時に乙が一切免責されない |
| E5 | 代替措置（減額・解除） | 代替措置の規定なし |

## F. 損害賠償

| ID | チェック項目 | RED基準 |
|----|-------------|---------|
| F1 | 損害賠償条項の有無 | 損害賠償条項なし |
| F2 | 賠償上限額 | 上限なし（無限責任） |
| F3 | 間接損害の除外 | 間接損害・逸失利益の除外なし |
| F4 | 故意・重過失の例外 | 故意・重過失も上限適用 |
| F5 | 賠償上限の定義の明確性 | 「委託料」の定義が曖昧 |

## G. 機密保持

| ID | チェック項目 | RED基準 |
|----|-------------|---------|
| G1 | 機密情報の定義 | 定義が広すぎる（口頭開示の取扱いなし） |
| G2 | 例外事由（公知、独自開発等） | 標準的な例外事由が欠落 |
| G3 | 存続期間 | 無期限 |
| G4 | 返却・破棄義務 | 返却/破棄の手続きが不明確 |
| G5 | 法令に基づく開示 | 法令開示の例外なし |

## H. 解除・終了

| ID | チェック項目 | RED基準 |
|----|-------------|---------|
| H1 | 解除事由 | 一方的な任意解除権（催告なし） |
| H2 | 催告期間 | 催告期間なし、または極端に短い |
| H3 | 存続条項 | 存続条項なし、または不十分 |
| H4 | 反社条項 | 反社条項なし |
| H5 | 中途解約（準委任の場合） | 解約通知期間なし |
