# 推奨条項テンプレート

業務委託基本契約書・NDAの頻出修正パターン。
条項テキストはそのまま赤入れに使用可能。

---

## 1. 契約類型の選択（第4条追加項）

```
個別契約においては、当該業務の契約類型（請負または準委任）を明記するものとする。
契約類型の記載がない場合は、成果物の納入を伴う業務は請負型、
それ以外の業務は準委任型とみなす。
```

## 2. 検査期間（第5条）

```
甲は、前項により納入を受けた「本件成果物」について、納入後10営業日以内に
これを検査し、合否の結果を乙に通知するものとする。尚、上記期間内に甲より
乙に何らの通知もない場合、当該期間の満了の時点で「本件成果物」は
甲の検査に合格したものとみなす。
```

**ポイント**: 「5日」→「10営業日」。暦日ではなく営業日で計算。

## 3. 契約不適合 — 甲修正時の免責限定

```
「本件成果物」が納入された後、「本件成果物」が甲により修正または変更された
場合には、「本件成果物」に契約不適合が発見されたときであっても、
当該契約不適合が甲の修正又は変更に起因する場合に限り、
乙は当該契約不適合について責任を負わないものとする。
```

**ポイント**: 原文の「一切の責任を負わない」を「起因する場合に限り」に変更。

## 4. 著作権 — 乙帰属+甲ライセンス方式

JQITが乙（受託者）になることが多い場合に推奨。

### §1 著作権帰属
```
「本件業務」遂行の過程で乙が新たに作成した著作物（ソースコード、ドキュメント等を
含む。以下「本件著作物」という。）に係る著作権（著作権法第27条および第28条に
規定する権利を含む。）は、乙に帰属するものとする。ただし、個別契約において
著作権の譲渡を別途定めた場合はこの限りでない。
```

### §2 甲へのライセンス
```
乙は、甲に対し、「本件著作物」を「本件成果物」の利用目的の範囲内で使用、
複製および改変する非独占的かつサブライセンス不能のライセンスを許諾する。
当該ライセンスの対価は委託料に含まれるものとする。
```

### §3 既存著作物
```
乙が本件業務の着手前から保有していた著作物、汎用的なライブラリ、フレームワーク、
ツール等（以下「既存著作物等」という。）の著作権は乙に留保される。乙は、甲に対し、
当該既存著作物等を「本件成果物」の利用に必要な範囲で使用する非独占的かつ
譲渡不能のライセンスを許諾する。
```

### §4 産業財産権
```
「本件業務」遂行の過程で得られた発明・考案・意匠の創作について産業財産権を
取得する権利の帰属は、個別契約において定めるものとする。
個別契約に定めのない場合は乙に帰属する。
```

### §5 AI生成物
```
乙がAIツール（生成AI、コード補完ツール等を含む。）を利用して生成した成果物の
著作権の帰属は、本条第1項に準ずるものとする。乙は、AIツールの利用に関して
第三者の知的財産権を侵害しないよう合理的な注意を払うものとする。
```

### §6 残留知識
```
甲は、乙が「本件業務」を通じて蓄積した一般的な知識、技能およびノウハウを、
甲以外の第三者に対する同種の業務に使用することを妨げないものとする。
```

### §7 著作者人格権
```
乙は、「本件著作物」に関する著作者人格権を甲に対して行使しないものとする。
```

## 5. 対価 — 請負/準委任二本立て

### 請負型
```
請負型の個別契約の場合、甲は、乙に対して、個別契約に定める金額を、
第5条第4項に基づき「本件業務」が完了した日の属する月の翌月末日までに
支払うものとする。
```

### 準委任型
```
準委任型の個別契約の場合、甲は、乙に対して、個別契約に定める単価および
稼働時間に基づき算定した金額を、毎月末日締め翌月末日払いにて支払うものとする。
稼働時間の精算方法（固定型、実績精算型、上下限精算型等）は
個別契約において定めるものとする。
```

### 振込
```
前二項の支払は、甲が対価を別途乙が指定する銀行口座に振り込むことにより
行うものとする。振込手数料は甲の負担とする。
```

## 6. 準委任型の業務遂行

```
準委任型の個別契約において、乙は、善良な管理者の注意義務をもって
「本件業務」を遂行し、毎月末日までに当月の業務遂行状況を
甲に書面にて報告するものとする。
```

```
準委任型の個別契約は、甲または乙の一方が相手方に対し1ヶ月前までに
書面にて通知することにより、将来に向かって解約することができるものとする。
ただし、解約時点までに遂行された業務に対する対価は、
前条に基づき精算するものとする。
```

```
準委任型の個別契約において、乙は「本件業務」の遂行について結果の完成を
保証するものではなく、業務遂行過程における善管注意義務の履行をもって
その責任を果たすものとする。
```

## 7. 損害賠償

### §1 賠償範囲
```
甲または乙が本契約もしくは個別契約に違反し、相手方に損害を与えた場合、
当該違反当事者は、相手方に対し、当該違反と相当因果関係のある直接かつ
現実に生じた通常損害に限り、これを賠償する責任を負うものとする。
```

### §2 賠償上限
```
前項に基づく損害賠償の累計額は、当該損害の原因となった個別契約に基づき
甲が乙に支払った（または支払うべき）委託料の総額を上限とする。
ただし、故意または重過失による場合はこの限りでない。
```

### §3 間接損害除外
```
甲および乙は、本契約または個別契約に関連して、逸失利益、間接損害、
特別損害、偶発的損害または結果的損害については、予見の有無にかかわらず
相互に請求しないものとする。
```

## 8. 存続条項（推奨版）

```
第8条の規定については本契約の終了または満了後1年間、
第9条ないし第12条、第16条および第19条の規定については
本契約の終了または満了後も引き続き有効とし、
第15条の規定については本契約の終了または満了後5年間に限り有効とする。
```

## 9. NDA — 口頭開示の要件

```
ただし、口頭または視覚的手段により開示された情報については、
開示時に機密である旨を明示し、かつ開示後10営業日以内に書面にて
機密指定を確認したものに限り、機密情報として取り扱うものとする。
```

## 10. NDA — 残留知識条項

```
前項の規定にかかわらず、甲または乙の役員もしくは従業員が本件業務の
遂行過程で記憶に留めた一般的な知識、経験、技能およびノウハウ
（特定の機密情報そのものを意図的に利用する場合を除く。）については、
本契約による制限の対象外とする。
```

## 11. NDA — 損害賠償（限定版）

```
甲または乙が本契約に違反した場合には、当該違反当事者は、
当該違反行為と相当因果関係のある直接かつ現実に生じた通常損害
（合理的な弁護士費用を含む。）に限り、相手方に対し賠償するものとする。
ただし、故意または重過失による場合はこの限りでない。
```
