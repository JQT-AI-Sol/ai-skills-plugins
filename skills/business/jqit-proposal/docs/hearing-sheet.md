# ヒアリングシート（初回商談用）

初回の商談で確認すること。★は必ず聞く。全部埋まらなくてOK。

> **2回目以降のヒアリング（デモを見せた後など）には、別のシートを使う。**
> → [`ai-solution-demo` のヒアリングガイド](../../ai-solution-demo/references/hearing-guide.md)

---

## 使い方

1. 商談前にこのシートを手元に用意しておく（印刷 or 画面表示）
2. 商談中にメモを取りながらチェック
3. 商談後に「商談メモ」として清書する（テンプレートは末尾に掲載）

---

## 1. 基本情報（商談前に書いておく）

| 項目 | 記入欄 | 必須 |
|------|--------|:---:|
| 会社名 | | ★ |
| 担当者の名前・役職・部署 | | ★ |
| 連絡先（メール / 電話） | | ★ |
| 業種 | | ★ |
| どこから来た相談か（紹介？問い合わせ？） | | |

---

## 2. 何に困っているか

**ゴール: お客様が何に困っていて、なぜ今なのかを掴む**

| 質問 | 記入欄 | 必須 |
|------|--------|:---:|
| 今回の相談を一言で言うと？ | | ★ |
| 具体的にどんな仕事で困っているか？ | | ★ |
| その仕事は今どうやっているか？ | | ★ |
| 一番の問題は？（時間がかかる / お金がかかる / ミスが多い / 特定の人しかできない） | | ★ |
| なぜ今このタイミングで検討しているか？ | | |
| 以前に似た取り組みをしたことはあるか？ | | |

**もっと掘り下げたいときの質問**:
- 「その仕事に関わっている人は何人くらいですか？」
- 「月にどのくらいの件数を処理していますか？」
- 「1件あたりどのくらい時間がかかりますか？」
- 「ミスが起きたときの影響はどの程度ですか？」

---

## 3. どうなりたいか

| 質問 | 記入欄 | 必須 |
|------|--------|:---:|
| 理想的にはどういう状態にしたいか？ | | ★ |
| 最低限ここだけは解決したい、というポイントは？ | | ★ |
| 成功したかどうか、何で判断するか？（かかる時間が半分になる、等） | | |
| 「これは要らない」というものはあるか？ | | |

---

## 4. 今のシステム・技術まわり

| 質問 | 記入欄 | 必須 |
|------|--------|:---:|
| 今使っているシステムやツールは？ | | ★ |
| データはどこに入っているか？（Excel / 社内システム / クラウドサービス等） | | |
| 社内にシステムに詳しい人やIT部門はいるか？ | | |
| セキュリティの制約はあるか？（「社外のサーバーはNG」等） | | |
| 他のシステムとつなぐ必要はあるか？ | | |
| このシステムを使う人は何人くらいか？ | | |

---

## 5. お金・スケジュール・進め方

| 質問 | 記入欄 | 必須 |
|------|--------|:---:|
| いつまでに使い始めたいか？ | | ★ |
| 予算のめどはあるか？（ざっくりでOK） | | |
| 予算はもう確保済みか？これから申請するか？ | | |
| 最終的にOKを出すのは誰か？ | | ★ |
| OKを出すまでの流れは？（担当者が決める / 上長の承認がいる / 社内稟議が必要） | | |
| 他社にも相談しているか？ | | |
| 選ぶときに何を重視するか？（価格 / スピード / 技術力 / 実績） | | |

---

## 6. 商談の終わりに確認すること

- [ ] 今日話した内容の認識が合っているか確認した
- [ ] **次回の打ち合わせ日を決めた（仮でもOK）**
- [ ] JQIT側の宿題を確認した（提案書 / 見積もり / デモ 等）
- [ ] 先方側の宿題を確認した（資料提供 / 社内確認 等）
- [ ] それぞれの期限を決めた

```markdown
-

1.
2.
3.

## 条件

- 時期:
- 決める人:
- 他社への相談:

## 提案やデモへの反応

- 違った点:

```

## 聞きにくいことの切り出し方

| 聞きたいこと | こう聞くとスムーズ |
|------------|----------------|
| 予算 | 「ざっくりで構わないのですが、ご予算感はありますか？規模感だけでも伺えると、最適なご提案ができます」 |
| 決める人 | 「ご導入の際はどなたがご判断されますか？次回はその方にもご同席いただけると嬉しいのですが」 |
| 他社状況 | 「他のベンダーさんともお話しされていますか？差別化ポイントを意識してご提案したいので」 |
